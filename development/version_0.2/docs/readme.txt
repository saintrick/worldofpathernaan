================================================
=================Installation===================
================================================
1. Upload all your files to your web root
2. Edit inc/config.php with your MySQL info and be sure to change 'SESSION_KEY' to something random.
3. goto phpMyAdmin and run docs/database.sql on the database you created.
4. Register your admin account and edit the row in phpMyAdmin and set the column 'rank' to '3'
